# Mundhe Banni Discourse Theme

This theme makes `https://mundebanni-community.discourse.group` visually match the Mundhe Banni app.

## What It Changes

- Black app-style header
- Orange primary actions
- Mundhe Banni blue text color
- White card surfaces
- 8px rounded cards and controls
- Softer topic list, category, tag, composer, modal, and sidebar styling
- Mobile refinements for topic lists and composer

## Install In Discourse

1. Open Discourse admin.
2. Go to `Admin → Customize → Themes`.
3. Click `Install`.
4. Import this folder as a theme, or push this folder to a GitHub repo and install from the repo URL.
5. Preview the theme first.
6. Set it as the default theme when it looks correct.

## Recommended Color Scheme

Create a color scheme in `Admin → Customize → Colors` with these values:

| Token | Color |
| --- | --- |
| Primary | `#1D2B53` |
| Secondary | `#FFFFFF` |
| Tertiary | `#E58A2B` |
| Quaternary | `#1A7A4A` |
| Header Background | `#050505` |
| Header Primary | `#FFFFFF` |
| Highlight | `#FFF4E6` |
| Danger | `#C84B2F` |
| Success | `#1A7A4A` |
| Love | `#E58A2B` |

## Logo

Upload the Mundhe Banni logo in:

`Admin → Settings → Branding`

Use a white logo version for the dark header.

## Safe Rollback

If the theme causes an issue, open:

`https://mundebanni-community.discourse.group/safe-mode`

Then disable the theme from admin.
